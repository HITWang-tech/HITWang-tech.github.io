## 1 卷积神经网络
本章介绍卷积神经网络（CNN）的基本原理、构成组件（卷积层、填充、步幅、汇聚层、多通道）及经典模型LeNet，阐述其在图像处理中的优势和应用。

### 1.1 从全连接层到卷积
全连接层处理图像**参数量爆炸**，且破坏空间结构；CNN利用**平移不变性**与**局部性**，大幅减少参数。

#### 1.1.1 平移不变性 + 局部性
- 平移不变性：同一特征在不同位置应被同样识别。
- 局部性：计算仅依赖输入局部区域。

满足两原则后，得到**离散卷积**（实际为互相关）：

$$
[H]_{i,j}=u+\sum_{a=-\Delta}^\Delta\sum_{b=-\Delta}^\Delta [v]_{a,b}[X]_{i+a,j+b}
$$

其中 $$v$$ 为**卷积核/滤波器**。

#### 1.1.2 卷积与互相关
卷积：

$$
(f*g)(i,j)=\sum_a\sum_b f(a,b)g(i-a,j-b)
$$

数学上卷积与互相关运算类似，区别在于卷积需翻转核函数。实际应用中因卷积核可学习，互相关运算常被称为卷积运算。

#### 1.1.3 通道
图像含多个通道（如RGB三通道），输入输出均为三维张量（高度、宽度、通道）。卷积核需考虑输入通道，隐藏表示通过多通道堆叠形成特征映射，不同通道可识别不同特征。

输入为 $$h×w×c_{in}$$
输出为 $$h×w×c_{out}$$
卷积核形状：

$$
c_{out} × c_{in} × k_h × k_w
$$

三维张量：

$$
[H]_{i,j,d}=\sum_{a,b}\sum_c [V]_{a,b,c,d}[X]_{i+a,j+b,c}
$$

### 1.2 图像卷积
#### 1.2.1 互相关运算
卷积层核心计算，输入张量与卷积核窗口按元素相乘求和。输出大小为输入大小 $$(n_h  × n_w )$$ 减去卷积核大小 $$(k_h  × k_w )$$ 加1，即 $$(n_h - k_h + 1) × (n_w - k_w + 1)$$ 通过corr2d函数实现，如3×3输入与2×2核运算得2×2输出。
```python
import torch
from torch import nn
from d2l import torch as d2l

def corr2d(X, K):
    h, w = K.shape
    Y = torch.zeros((X.shape[0]-h+1, X.shape[1]-w+1))
    for i in range(Y.shape[0]):
        for j in range(Y.shape[1]):
            Y[i,j] = (X[i:i+h, j:j+w] * K).sum()
    return Y
```
#### 卷积层
包含可训练的卷积核权重和标量偏置，前向传播为互相关运算加偏置。用corr2d函数实现，可通过训练学习卷积核参数。
```python
class Conv2D(nn.Module):
    def __init__(self, kernel_size):
        super().__init__()
        self.weight = nn.Parameter(torch.rand(kernel_size))
        self.bias = nn.Parameter(torch.zeros(1))
    def forward(self, x):
        return corr2d(x, self.weight) + self.bias
```

#### 图像中目标的边缘检测
构造6×8黑白图像（中间四列黑色）和1×2卷积核[1.0, -1.0]，互相关运算可检测垂直边缘，输出1代表白到黑边缘，-1代表黑到白边缘，转置图像后垂直边缘消失。
```python
X = torch.ones((6,8))
X[:,2:6] = 0
K = torch.tensor([[1.0, -1.0]])
Y = corr2d(X, K)
```

#### 学习卷积核
使用内置二维卷积层，初始化随机卷积核，通过比较输出与目标的平方误差，梯度下降更新核参数。10次迭代后，学习到的卷积核接近预设边缘检测核。
```python
# 构造一个二维卷积层，它具有1个输出通道和形状为（1，2）的卷积核 
conv2d = nn.Conv2d(1,1, kernel_size=(1, 2), bias=False) 

# 这个二维卷积层使用四维输入和输出格式（批量大小、通道、高度、宽度）， 
# 其中批量大小和通道数都为1 
X = X.reshape((1, 1, 6, 8)) 
Y = Y.reshape((1, 1, 6, 7)) 
lr = 3e-2 # 学习率 


for i in range(10): 
    Y_hat = conv2d(X) 
    l = (Y_hat - Y) ** 2 
    conv2d.zero_grad() 
    l.sum().backward() 
    # 迭代卷积核 
    conv2d.weight.data[:] -= lr * conv2d.weight.grad 
    if (i + 1) % 2 == 0: 
        print(f'epoch {i+1}, loss {l.sum():.3f}')
```

#### 互相关和卷积
严格卷积需翻转核函数，因卷积核从数据学习，实际应用中互相关运算效果与卷积一致，深度学习文献常称互相关为卷积。

#### 特征映射和感受野
输出的卷积层称为特征映射，某元素的感受野是前向传播中影响其计算的所有先前层元素。深层网络可扩大感受野，检测更广区域特征。

### 填充和步幅
#### 填充（padding）
在输入图像边界填充元素（通常为0），以增加输出高度和宽度，减少边缘像素信息丢失。
输出形状：
$$
(n_h -k_h +p_h +1) × (n_w -k_w +p_w +1)
$$
常设置$$p_h=k_h-1$$、$$p_w=k_w-1$$使输入输出大小相同。卷积核大小多为奇数，便于对称填充。
```python
conv2d = nn.Conv2d(1,1,kernel_size=3,padding=1)
```

#### 步幅（stride）
卷积窗口滑动的元素数量，可缩减输出空间维度。

输出形状：

$$
\lfloor (n_h -k_h +p_h +s_h)/s_h \rfloor × \lfloor (n_w -k_w +p_w +s_w)/s_w \rfloor
$$

当输入尺寸能被步幅整除且填充为 $$k_h-1$$、 $$k_w-1$$时，输出尺寸为 $$(n_h/s_h)×(n_w/s_w)$$
```python
conv2d = nn.Conv2d(1,1,kernel_size=3,padding=1,stride=2)
```

### 多输入多输出通道
#### 多输入通道
输入含多个通道时，每个通道与对应卷积核互相关后求和得到输出。如2输入通道与2个2×2核运算，输出为各通道结果之和。
```python
def corr2d_multi_in(X, K):
    return sum(corr2d(x,k) for x,k in zip(X,K))
```

#### 多输出通道
为每个输出通道创建$$c_i×k_h×k_w$$的卷积核，卷积核形状为$$c_o×c_i×k_h×k_w$$

各输出通道独立计算所有输入通道的互相关并叠加。
```python
def corr2d_multi_in_out(X, K):
    return torch.stack([corr2d_multi_in(X,k) for k in K], 0)
```

#### 1×1 卷积
卷积核大小1×1，仅在通道维度计算，可视为每个像素位置的全连接层，实现通道维度变换，权重维度为$$c_o×c_i$$，需偏置。

---

### 汇聚层（池化）
#### 最大汇聚层和平均汇聚层
最大汇聚层输出窗口内最大值，平均汇聚层输出窗口内平均值。可减轻卷积层对位置的过度敏感，常用2×2窗口、步幅2，减少空间维度。
#### 填充和步幅
汇聚层可设置填充和步幅调整输出形状，默认步幅与窗口大小相同。如3×3窗口、填充1、步幅2，4×4输入输出2×2。
#### 多个通道
汇聚层在每个输入通道上单独运算，输出通道数与输入通道数相同。

```python
pool2d = nn.MaxPool2d(3, padding=1, stride=2)
```

### LeNet
最早成功的CNN，用于手写数字识别。

#### 结构
由卷积编码器（两个卷积层）和全连接层密集块（三个全连接层）组成。
- 卷积块含卷积层、sigmoid激活函数和平均汇聚层
- 第一卷积层6输出通道、5×5核、填充2
- 第二卷积层16输出通道、5×5核
- 平均汇聚层2×2窗口、步幅2；
- 全连接层输出120、84、10维，展平后输入。

```python
net = nn.Sequential(
    nn.Conv2d(1,6,kernel_size=5,padding=2), nn.Sigmoid(),
    nn.AvgPool2d(kernel_size=2,stride=2),
    nn.Conv2d(6,16,kernel_size=5), nn.Sigmoid(),
    nn.AvgPool2d(kernel_size=2,stride=2),
    nn.Flatten(),
    nn.Linear(16*5*5, 120), nn.Sigmoid(),
    nn.Linear(120, 84), nn.Sigmoid(),
    nn.Linear(84, 10)
)
```

#### 训练（GPU版）
使用Fashion-MNIST数据集，batch_size=256，GPU加速训练。
初始化权重用Xavier均匀分布，优化器为SGD，学习率0.9，交叉熵损失函数。
```python
def train_ch6(net, train_iter, test_iter, num_epochs, lr, device):
    def init_weights(m):
        if type(m)==nn.Linear or type(m)==nn.Conv2d:
            nn.init.xavier_uniform_(m.weight)
    net.apply(init_weights)
    net.to(device)
    optimizer = torch.optim.SGD(net.parameters(),lr=lr)
    loss = nn.CrossEntropyLoss()
    # 训练循环（省略动画）
    for epoch in range(num_epochs):
        net.train()
        for X,y in train_iter:
            X,y = X.to(device),y.to(device)
            l = loss(net(X),y)
            optimizer.zero_grad()
            l.backward()
            optimizer.step()

lr, epochs = 0.9, 10
train_ch6(net, train_iter, test_iter, epochs, lr, device)
```
### 本章小结
卷积神经网络通过卷积层、填充、步幅、汇聚层和多通道等组件，有效利用图像空间结构，减少参数数量，在计算机视觉任务中性能优异。LeNet作为早期经典模型，证明了CNN在手写数字识别等任务的有效性，为后续复杂网络架构奠定基础，其设计思想（卷积块提取特征、全连接层分类）至今仍被广泛应用。
## 现代卷积神经网络
### 深度卷积神经网络（AlexNet）
#### 学习表征
2012年前，图像特征多为机械计算，如SIFT、SURF等。而AlexNet提出特征应被学习，且由多个共同学习的神经网络层组成。其在2012年ImageNet挑战赛中表现轰动，底层学习到类似传统滤波器的特征抽取器，更高层逐步表示更大特征直至整个物体。

数据方面，深度模型需大量有标签数据，ImageNet数据集的出现（2009年发布，含百万样本、千种类别）改变了这一状况；硬件方面，GPU的高吞吐量矩阵和向量乘法运算与卷积层计算相似，AlexNet使用两个3GB显存的NVIDIA GTX580 GPU实现快速卷积运算，推动了深度学习发展。
#### AlexNet
由八层组成（五个卷积层、两个全连接隐藏层、一个全连接输出层），使用ReLU激活函数。模型设计上，第一层卷积窗口11×11，步幅4，后续卷积窗口逐渐减小，部分卷积层后有最大汇聚层，卷积通道数是LeNet的10倍。通过暂退法控制全连接层复杂度，训练时进行图像增强扩充数据。
### 使用块的网络（VGG）
#### VGG块
由一系列卷积层（3×3卷积核、填充1以保持分辨率）和用于空间下采样的最大汇聚层（2×2窗口、步幅2）组成。函数vgg_block实现该块，参数包括卷积层数量、输入和输出通道数。
#### VGG网络
分为卷积层和全连接层两部分。通过conv_arch变量指定每个VGG块的卷积层个数和输出通道数，原始VGG-11有5个卷积块（前两各1层，后三各2层），输出通道数从64翻倍至512，全连接模块与AlexNet相同。
### 网络中的网络（NiN）
#### NiN块
以普通卷积层开始，后接两个1×1卷积层（充当逐像素全连接层，带ReLU激活函数），第一层卷积窗口形状用户设置，后两层固定1×1。
#### NiN模型
使用与AlexNet相同的卷积层窗口形状和输出通道数，每个NiN块后有最大汇聚层（3×3窗口、步幅2）。取消全连接层，用输出通道数等于标签类别数的NiN块和全局平均汇聚层生成对数几率，减少了模型参数。
### 含并行连结的网络（GoogLeNet）
#### Inception块
由四条并行路径组成，前三条用1×1、3×3、5×5卷积层，中间两条路径先1×1卷积降通道数，第四条用3×3最大汇聚层后接1×1卷积。各路径合适填充使输入输出高宽一致，最后通道维连结输出，超参数为每层输出通道数。
#### GoogLeNet模型
使用9个Inception块和全局平均汇聚层堆叠，块间有最大汇聚层降维。第一个模块用7×7卷积层，之后各模块包含Inception块和汇聚层等，最后接全局平均汇聚层和全连接层输出。
### 批量规范化
#### 训练深层网络
数据预处理影响大，需标准化输入；中间层变量分布偏移可能阻碍收敛；深层网络易过拟合。批量规范化在训练迭代中先标准化输入（减均值除标准差，基于当前小批量），再应用比例系数和偏移，训练和预测模式功能不同。
#### 批量规范化层
全连接层中置于仿射变换和激活函数间；卷积层中在卷积后激活前应用，对每个输出通道执行，有自己的拉伸和偏移参数。预测时用移动平均估算整个训练数据集的均值和方差。
#### 从零实现
定义batch_norm函数处理训练和预测模式下的标准化，创建BatchNorm层保存拉伸、偏移参数及均值和方差的移动平均值。
#### 使用批量规范化层的LeNet
在卷积层或全连接层之后、激活函数之前应用批量规范化层，训练时可使用更大学习率。
### 残差网络（ResNet）
#### 函数类
假设神经网络架构F，若理想函数f∗∈F可训练得到，否则需设计更强大的F′。残差网络核心思想是让附加层易于包含原始函数，通过残差块实现，输入可跨层传播。
#### 残差块
有两个相同输出通道数的3×3卷积层，各后接批量规范化层和ReLU激活函数，输入通过跨层数据通路加在最后的ReLU前。需卷积层输出与输入形状一样，改变通道数时用1×1卷积层变换。
#### ResNet模型
前两层为7×7卷积层和3×3最大汇聚层（带批量规范化），后接4个残差块模块，每个模块用若干残差块，第一个模块通道数与输入一致，之后模块第一个残差块翻倍通道数并减半高宽，最后接全局平均汇聚层和全连接层。
### 稠密连接网络（DenseNet）
#### 从ResNet到DenseNet
ResNet将函数展开为 $$f(x)=x+g(x)$$ ，DenseNet输出是连接而非相加，实现从x到其展开式的映射，最后结合到多层感知机减少特征数量。
#### 稠密块体
由多个卷积块组成，每个卷积块输出通道数相同，前向传播时将每个卷积块的输入和输出在通道维连结，卷积块通道数称为增长率。
#### 过渡层
通过1×1卷积层减小通道数，用步幅为2的平均汇聚层减半高和宽，控制模型复杂度。
#### DenseNet模型
先使用单卷积层和最大汇聚层，后接4个稠密块，块间用过渡层控制通道数和高宽，最后接全局汇聚层和全连接层输出。
### 本章小结
#### 关键发现
本章介绍的现代卷积神经网络架构（AlexNet、VGG、NiN、GoogLeNet、ResNet、DenseNet等）通过不同的设计理念和结构改进，推动了计算机视觉领域的发展。如AlexNet证明学习特征可超越手工设计特征；VGG使用可复用卷积块简化网络设计；ResNet通过残差块解决深层网络训练困难问题，使训练深层网络成为可能；DenseNet通过稠密连接进一步提升特征利用效率。
#### 领域重要性
这些模型为后续深度学习研究提供了重要的架构模板和设计思想，如块的使用、批量规范化、残差连接等技术被广泛应用于各种深度学习任务中，显著提高了模型的性能和训练效率，对计算机视觉及相关领域的发展产生了深远影响。
